Ready-to-run commands

          

            

              
Daily automation

              python -m automation.main --config config.json
              
Use in cron or Task Scheduler to produce and upload your daily batch.

              
            

            

              
Dry run

              python -m automation.main --config config.json --dry-run
              
Generate everything without uploading to review the visuals and narration.

              
            

            

              
Manual topic

              python -m automation.main --config config.json --topic "Space Tourism"
              
Override the automated topic selection for timely campaigns or partnerships.
